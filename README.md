# LvlUp - Productivity App

Este proyecto utiliza el stack **FARM** (FastAPI, React, MongoDB).

## Estructura del Proyecto

*   **`frontend/`**: Aplicación Next.js (El código original).
    *   Ejecutar: `cd frontend` -> `npm run dev`
*   **`backend/`**: Servidor API en Python/FastAPI.
    *   **Git Bash / Bash**: `cd backend` → `source venv/Scripts/activate` → `uvicorn main:app --reload`
    *   **PowerShell**: `cd backend` → `.\venv\Scripts\Activate.ps1` → `uvicorn main:app --reload`
    *   **CMD**: `cd backend` → `venv\Scripts\activate` → `uvicorn main:app --reload`

## Configuración Manual
Consulta las guías específicas dentro de cada carpeta para detalles de instalación.
